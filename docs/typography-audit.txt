ALTA GROUP TYPOGRAPHY AUDIT
============================

Scope: Full codebase review — global CSS, font imports, Tailwind v4 theme, design
tokens, all route areas, shadcn primitives, mobile/dark mode.

Date: June 22, 2026
Status: Audit only — no code changes implemented.


EXECUTIVE SUMMARY
-----------------

The site has a strong intended three-font system (Fraunces / Inter Tight /
JetBrains Mono) defined in src/styles.css and imported in src/routes/__root.tsx.
In practice, that system is partially adopted: ~70 routes use PageShell with
tokenized heroes, but 600+ arbitrary pixel sizes and 550+ inline font-mono
combinations bypass the design tokens.

Two competing EmptyState components, an undefined type-finance-nums class, and
unused formFieldLabelClass are the clearest signs the system is not yet unified.

Verdict: Premium intent is visible (especially landing, auth, private banking,
governance). Consistency is moderate — readable and institutional in places,
fragmented in forms, tables, empty states, and sub-navs.

6. RECOMMENDED FINAL TYPOGRAPHY SYSTEM
--------------------------------------

One unified system — keep the existing three fonts, enforce tokens:

Role                    Font / spec
Display serif           Fraunces 400 — type-display for page heroes;
                        type-display-governance for about/governance only; no custom clamps
UI sans                 Inter Tight — body 15px/1.6 (type-body); small 13px (type-body-sm);
                        nav 13px (type-nav); subnav 12px (type-subnav)
Mono / data             JetBrains Mono 500 — all uppercase labels, eyebrows, section titles, table headers
Financial figures       JetBrains Mono 600 + tabular — <Florin /> everywhere;
                        type-finance-sm/md/lg/xl/hero by context;
                        add type-finance-nums as alias of type-finance-md
Buttons (default)       Inter Tight 500 — 13px (tokenize as type-button)
Buttons (institutional) JetBrains Mono 500 — 11px uppercase 0.20em — ops/admin only
Labels / eyebrows       JetBrains Mono 500 — 10px / 0.16em uppercase (type-ui-label, type-meta, type-eyebrow)
Tables                  Headers: mono 10px 0.16em; body: sans 13px; amounts: mono tabular — mandatory .alta-table
Forms                   Labels: type-ui-label; inputs: Inter Tight 14px (text-sm);
                        help text: type-body-sm muted; override shadcn Label


TRACKING SCALE (recommended — collapse to 3 values)
---------------------------------------------------

0.16em — labels, meta, table headers, form labels
0.20em — eyebrows, institutional buttons, footer column titles
0.22em — wordmark / brand lockups only (AltaWordmark, login footer tags)

CONCLUSION
----------

The foundation is right for a premium financial brand. The work ahead is
enforcement: one empty state, one label pattern, one table pattern, one figure
component, and retiring arbitrary pixel typography in favor of the tokens already
defined in src/styles.css.
